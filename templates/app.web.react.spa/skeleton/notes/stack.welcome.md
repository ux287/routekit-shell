# Welcome to your React SPA

This project was scaffolded from the **app.web.react.spa** stack — a minimal
React + Vite + TypeScript + Tailwind single-page app with basic client-side routing.

- Run it: `npm install && npm run dev` → serves on localhost.
- Build your app: edit `src/pages/Home.tsx`, add pages under `src/pages/`, and register
  routes in `src/App.tsx`.
- UI primitives (button, card, input, label, separator) live under `src/components/ui/`.

Describe what you want to build and let the RouteKit dispatcher take it through the pipeline.
