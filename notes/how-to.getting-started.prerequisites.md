# Prerequisites for RouteKit Shell

## Required Software

### Node.js 18+

RouteKit Shell requires Node.js version 18 or higher for modern JavaScript features and optimal performance.

**Installation:**

- Download from [nodejs.org](https://nodejs.org/)
- Verify installation: `node --version`

### Git

Version control system for managing your project code.

**Installation:**

- Download from [git-scm.com](https://git-scm.com/)
- Verify installation: `git --version`

## Recommended Tools

### VS Code

Primary editor with excellent TypeScript support and extensions ecosystem.

**Why VS Code?**

- Built-in TypeScript support
- Excellent debugging capabilities
- Rich extension marketplace
- Claude Code integration

**Installation:**

- Download from [code.visualstudio.com](https://code.visualstudio.com/)

### Claude Code Extension

AI pair programming assistant that integrates with RouteKit's RAG system.

**Features:**

- Context-aware code suggestions
- Access to your project documentation via RAG
- MCP integration for advanced workflows

**Installation:**

- Install from VS Code marketplace
- Configure MCP servers (covered in RAG integration guide)

## Optional but Recommended

### GitHub/GitLab Account

For version control hosting and collaboration.

### Vercel/Netlify Account

For easy deployment of your RouteKit applications.

## System Requirements

### Minimum Requirements

- **RAM:** 4GB minimum, 8GB recommended
- **Storage:** 1GB free space for Node.js dependencies
- **OS:** macOS, Windows 10+, or Linux (Ubuntu 18+)

### Development Environment

- Terminal/Command Prompt access
- Internet connection for package installation
