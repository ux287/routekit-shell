## Philosophy

RouteKit Shell is designed for rapid prototyping and AI-assisted development. Every feature is built to:

- **Accelerate Development**: Get from idea to working prototype in minutes
- **Enable AI Workflows**: Built-in RAG and MCP integration for Claude Code
- **Maintain Quality**: TypeScript, testing, and linting out of the box
- **Scale Gracefully**: Start simple, add complexity as needed
